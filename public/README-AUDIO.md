# Audio Files for Tirupati Trip Application

## Required Audio File

To complete the Blessings page experience, you need to add the following audio file:

### `om-chants.mp3`
- **Location**: Place this file directly in the `/public` folder
- **Purpose**: Sacred Om chanting for the Blessings page
- **Format**: MP3 audio file
- **Recommended**: 2-5 minutes of continuous Om chanting or Venkateswara slokas

## How to Add the Audio File

1. **Find or Record**: Get an Om chanting or Venkateswara sloka MP3 file
2. **Rename**: Ensure the file is named exactly `om-chants.mp3`
3. **Place**: Put the file in the `public` folder of your project
4. **Test**: Visit the Blessings page to hear the audio

## Audio Sources Priority

The application will try to load audio in this order:
1. `/om-chants.mp3` (local file - highest priority)
2. Online bell sound (fallback)
3. Online sample music (fallback)

## Recommended Audio Content

- **Om Chanting**: Continuous "Om" or "Aum" chanting
- **Venkateswara Slokas**: Sacred hymns dedicated to Lord Venkateswara
- **Temple Bells**: Traditional temple bell sounds
- **Duration**: 2-5 minutes (will loop automatically)
- **Quality**: Clear, peaceful, and spiritually uplifting

## File Requirements

- **Format**: MP3
- **Size**: Recommended under 10MB for faster loading
- **Quality**: 128kbps or higher for good audio quality
- **Length**: 2-5 minutes (loops automatically)

Once you add the `om-chants.mp3` file, the Blessings page will automatically use it for the sacred audio experience.
